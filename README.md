# Quiz Application

## Overview
This Quiz Application is a simple web-based project developed as part of the TechnoHacks Internship. It allows users to answer multiple-choice questions and view their score at the end of the quiz.

## Features
- Multiple-choice questions
- Instant answer validation
- Score calculation
- Responsive user interface

## Technologies Used
- HTML
- CSS
- JavaScript
